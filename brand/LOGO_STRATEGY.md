# 人格麻將 Logo 策略与方案

> 本文档记录 logo 的完整思考过程、约束、方向与方案。
> 配套 SVG 概念稿见 `brand/concepts/`（4 方向 × 3 变体 = 12 个），可视化预览见 `brand/preview.html`。
> 状态：仅为提案，尚未落地到网站任何位置。日期：2026-07-17

---

## 1. 设计目标（需求）

1. **无文字**：一个中英（及未来其他语言）通用的单一 logo，不含任何汉字/字母。
2. **概念要灵活**：未来会有多种人格模型（不止 5 维，可能 4 维、6 维），设计**不能锁死在固定的形状数量 / 角数**（例如不用五边形）。
3. **必含 Ψ (Psi)**：心理学 / 心智的通用希腊字母与科学符号（明确要求）。
4. **贴合网站**：不要写实；风格要和现有网站的暖色、卡通/手绘卡牌风一致，能自然融进页面。

### 两个已认可的参考方向（作为出发点）
- **参考 A**：卡牌 + 脑 + Ψ（神经网络里长出 Ψ，带节点/连线，科技感）。
- **参考 B**：Ψ + **鲁宾花瓶错觉（Vase / 双侧脸）** + 扁平卡通风格。

---

## 2. 想放进 logo 的元素（元素池）

| 元素 | 代表含义 | 对应哪条目标 |
|---|---|---|
| **Ψ (Psi)** | 心理学 / 心智 | 硬需求③；语言无关① |
| **麻将牌身**（圆角矩形 + 象牙金边） | 游戏本体「麻将」；直接呼应网站卡框 | 贴合网站④ |
| **鲁宾花瓶 / 双侧脸** | 「先读懂自己，再读懂别人」+ 知觉双重性 | 参考 B；点游戏核心循环 |
| **圆形轨道 / 自由数量的点** | 多种人格模型（4/5/6 维皆可） | 灵活②（圆形≠多边形） |
| **神经节点 / 连线** | 数据、测评的科学性、脑力博弈 | 参考 A |
| **暖金 + 象牙 + 陶土 配色** | 与网站同源 | 贴合网站④ |

**关键洞察**
- **Ψ 本身就像一只高脚杯 / 花瓶**：所以「Ψ + 鲁宾花瓶」几乎天作之合——两侧补上对望侧脸，中间的 Ψ 立刻被读成花瓶。
- **用「圆形」承载维度**，无论 4/5/6 维都不突兀，天然绕开「锁死维度数」。

---

## 3. 与网站视觉的一致性（重点）

logo 必须看起来「本来就长在这个网站里」。对齐的现有元素：

| 网站现有元素 | 在 logo 里怎么呼应 |
|---|---|
| 卡框（象牙面 `#fdf8f1` + 金边 `#c39a52` + 圆角 + 刻蚀内框 `psy-etched`） | 方向1「麻将牌」直接复用这套框；变体 b 加内刻蚀细框 |
| 维度 chip（胶囊形，如 `O 3`、`fig-pill`） | 方向3 变体 b 的环上胶囊沿用同款样式 |
| 暖金主色 `--psy-accent #c39a52` / 深金 `#9a7448` | 所有 Ψ 用同一套金色渐变 |
| 米色背景 `--psy-bg #f4edd9` / 象牙面 `#fdf9f0` | logo 底色同源，放页面上能「消融」融入 |
| 陶土红 `--psy-danger #c9603f` / 柔绿 `#6f8f55` | 仅作点缀（侧脸、节点），不喧宾夺主 |
| 标题衬线 + 正文 Noto Sans SC | logo 为纯图形，与标题锁版时风格协调 |

> **不要**把 5 个维度色（O紫 C蓝 E黄 A绿 N红）搬进 logo——那会把 logo 和「5 维」绑死，违反「灵活维度」。维度色留给游戏内 UI。

---

## 4. 设计思路（怎么从元素走到方案）

1. **双层品牌系统**：一简一繁，各司其职，避免「既要小尺寸清晰又要信息丰富」的自相矛盾。
   - **主标记（语言无关 logo / icon）**：越简单越好 → favicon、App 图标、页首。
   - **徽记（emblem，讲故事）**：信息更丰富 → 加载页、结算分享、规则大图。
2. **扁平、少即是多**：粗描边、圆角、纯色块，保证 16–32px favicon 仍认得出 Ψ。写实和复杂神经网络在小尺寸会糊，弱化。
3. **暖不冷**：全部用网站的暖金 / 象牙 / 陶土，放弃参考稿里的冷蓝银。
4. **每个方向给多个变体**：同一概念做出「经典 / 更贴卡框 / 单色 / 极简 / 桥接」等不同取舍，方便对比取舍。

## 5. 四个方向 × 每方向三个变体（共 12 个）

> 全部见 `concepts/` 与 `preview.html`。

### 方向 1 · Psi Tile（麻将牌）— **最贴网站、推荐作主标记**
- **a 经典**（`d1-tile-classic`）：象牙金边牌 + 金 Ψ。最简、最耐缩放。
- **b 刻蚀框**（`d1-tile-etched`）：加内刻蚀细框 + 四角饰点，呼应 `psy-etched` 卡框。
- **c 金身镂空**（`d1-tile-emboss`）：金牌身 + 象牙镂空 Ψ（浮雕/反白），适合深色与单色。

### 方向 2 · Psi Vase（鲁宾花瓶）— 叙事最强，宜作徽记
- **a 圆徽**（`d2-vase-badge`）：陶土双脸 + 金 Ψ-花瓶。
- **b 单色金**（`d2-vase-mono`）：侧脸改柔金，适合单色 / 页首 / 深色。
- **c 装进牌**（`d2-vase-tile`）：花瓶双脸放进麻将牌框，桥接方向 1 与 2。

### 方向 3 · Psi Orbit（圆形轨道）— 最能表达「灵活维度」
- **a 虚线轨道**（`d3-orbit-dashed`）：环上小牌 + 圆点交替（数量示意）。
- **b 胶囊环**（`d3-orbit-pills`）：环上散布网站同款胶囊 chip，一致性最强，可做旋转动效。
- **c 极简**（`d3-orbit-min`）：Ψ + 三个绕行圆点，可缩到 favicon。

### 方向 4 · Psi Mind（脑 + 节点）— 参考 A 的暖色简化
- **a 圆徽脑弧**（`d4-mind-badge`）：顶部脑半球虚线弧 + 节点。
- **b 节点极简**（`d4-mind-nodes`）：Ψ 三叉尖接神经节点 + 连线，无外框。
- **c 脑轮廓**（`d4-mind-silhouette`）：柔和脑轮廓内嵌 Ψ，扁平不写实。

---

## 6. 推荐

- **主 logo（默认 / favicon / 页首）：方向 1 · Psi Tile（首选变体 a 经典或 b 刻蚀框）。** 最简、最耐缩放、最贴网站卡框。
- **徽记（加载页 / 结算分享 / 规则大图）：方向 2 · Psi Vase 或 方向 3 · Psi Orbit。**
  - 想要「故事感 / 高级感」→ 方向 2。
  - 想要「灵活维度 + 动效潜力」→ 方向 3。
- 方向 4 作为「参考 A 的暖色简化版」备选，便于交代那条思路也认真做过。

> 一句话提案：**Ψ 一定保留；主 logo 用最简的『Ψ + 麻将牌』保证到处都清晰，另配一个『Ψ 花瓶双脸』徽记讲故事。全部换成网页暖金色，不锁维度数。**

## 8. Do / Don't

**Do**：Ψ 永远居中可辨；圆形/牌形承载；暖色同源；先保证 16px favicon 认得出。
**Don't**：不写实（真手、真脑、3D 金属反光）；不用五边形/固定 N 角；不用冷蓝银；不在标记本体里放文字（页首可「标记 + 文字」锁版，文字不属于 logo 本体）；不把 5 个维度色搬进 logo。
