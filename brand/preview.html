提案 · 尚未落地网站

方向 1 · Psi Tile（麻将牌）推荐主标记

Ψ + 麻将牌，最贴网站卡框、最耐缩放 → favicon / App 图标 / 页首。

a · 经典
象牙金边牌 + 金 Ψ
b · 刻蚀框
内刻蚀细框，呼应 psy-etched
c · 金身镂空
金牌身 + 象牙镂空 Ψ（深色/单色）

方向 2 · Psi Vase（鲁宾花瓶）

两张对望侧脸中间的负空间 = 金 Ψ-花瓶。「读己读人」叙事最强 → 加载页 / 分享图。

a · 圆徽
陶土双脸 + 金 Ψ-花瓶
b · 单色金
柔金侧脸，宜单色/页首
c · 装进牌
花瓶双脸放进麻将牌框

方向 3 · Psi Orbit（圆形轨道）

中心 Ψ + 圆形轨道（不锁维度数）。变体 b 用网站同款胶囊 chip，一致性最强、可做旋转动效。

a · 虚线轨道
环上小牌 + 圆点交替
b · 胶囊环
环上散布网站同款 chip
c · 极简
Ψ + 三绕行点，可缩到 favicon

方向 4 · Psi Mind（脑 + 节点）

参考「脑 + Ψ + 节点」方向的暖色简化版；节点数量随意、不锁维度。

a · 圆徽脑弧
脑半球虚线弧 + 节点
b · 节点极简
Ψ 三叉尖接神经节点
c · 脑轮廓
柔和脑轮廓内嵌 Ψ
