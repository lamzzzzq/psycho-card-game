# brand/ —— 人格麻將 Logo 设计交付

老板要做 logo。这里是完整的思考过程、方案与放置建议（2026-07-17）。

## 文件
- **[LOGO_STRATEGY.md](./LOGO_STRATEGY.md)** —— 需求梳理、核心洞察、设计原则、四个方案、推荐、配色 token、Do/Don't。**先读这个。**
- **[PLACEMENT.md](./PLACEMENT.md)** —— logo 放网页哪里（按优先级）+ Next.js 落地提示。
- **[preview.html](./preview.html)** —— 四个方案的可视化预览（浏览器直接打开；含 favicon 小尺寸测试 + 亮/暗底 + 页首锁版示意）。
- **concepts/** —— 可用的 SVG 概念稿：
  - `psi-tile.svg` —— 方案 A · Psi Tile（**推荐主标记 / favicon**）
  - `psi-vase.svg` —— 方案 B · Psi Vase（鲁宾花瓶，徽记）
  - `psi-orbit.svg` —— 方案 C · Psi Orbit（圆形轨道，灵活维度）
  - `psi-mind.svg` —— 方案 D · Psi Mind（脑 + Ψ + 节点，老板方向 A 暖色简化版）

## 30 秒结论
- **Ψ 一定保留**（老板硬需求）；全部换成网站暖金色，不用冷蓝银。
- **主 logo = Psi Tile**（Ψ + 麻将牌）：最简、耐缩放、到处清晰 → favicon / App 图标 / 页首。
- **徽记 = Psi Vase 或 Psi Orbit**：讲故事 / 灵活维度 → 加载页 / 结算分享 / 规则大图。
- 先落 **favicon + PWA + 页首** 三处，露出最大、风险最低。

> 这些 SVG 是**方向概念稿**（手写代码），用于快速对齐方向；定稿后可交设计师或图生细化。
