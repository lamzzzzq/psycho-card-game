## 30 秒结论
- **Ψ 一定保留**（硬需求）；全部用网站暖金色，不用冷蓝银。
- **主 logo = Psi Tile**（Ψ + 麻将牌）：最简、耐缩放、最贴网站卡框 → favicon / App 图标 / 页首。
- **徽记 = Psi Vase 或 Psi Orbit**：讲故事 / 灵活维度 → 加载页 / 结算分享 / 规则大图。

> SVG 为**方向概念稿**（手写代码），用于快速对齐方向；定稿后可交设计师或图生细化。
