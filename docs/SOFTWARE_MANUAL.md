## 1. 软件概述

人格麻将教学系统是一款基于人格心理学（大五人格 / Big Five 五因素模型）的网页游戏化教学软件。系统以「麻将」式的抽牌、弃牌、碰牌、归档玩法为载体，将五个人格维度（开放性、尽责性、外向性、宜人性、神经质）的知识融入对战过程，帮助学生在游戏中学习和理解人格心理学概念，提升课堂互动性与学习效果。

系统主要包含以下功能模块：

- **人格测评**：学生完成问卷，系统计算个人五维人格得分并生成人格图谱。
- **单机对战**：学生与计算机（AI）对手进行卡牌对战。
- **联机对战**：2–4 名学生在线同桌实时对战。
- **教学引导**：内置交互式教程，引导新用户理解玩法。
- **数据统计**：面向教师，汇总班级测评与对战数据。

## 2. 运行环境与访问方式

- **运行平台**：本软件为网页（Web）应用，无需安装，通过浏览器访问。
- **支持的操作系统**：Windows、macOS、Android、iOS 等。
- **支持的浏览器**：Google Chrome、Microsoft Edge、Safari、Firefox 等现代主流浏览器。
- **访问地址（系统网址）**：`https://www.personalitiesmahjong.com`（下文统称"系统网址"）。
- **访问方式**：在浏览器地址栏输入系统网址，即可打开首页。

**操作步骤：**
1. 打开浏览器。
2. 在地址栏输入系统网址并回车。
3. 系统加载并显示首页。

## 4. 人格测评

### 4.1 输入学号

点击「开始测评」后，首先进入学号输入界面。学号用于记录测评结果，便于课堂统计。

**操作步骤：**
1. 在输入框中输入本人学号。
2. 点击「开始测评」按钮。
3. 系统校验学号；若该学号已有测评记录，会提示「恢复上次结果」或「重新测评」两个选项，按需选择。

【配图：输入学号界面】

### 4.2 作答问卷

进入问卷后，系统逐题展示人格测评题目。每道题为一句陈述，用户根据自身情况在「非常不同意 — 非常同意」的量表上选择。

**界面元素说明：**
- 顶部进度条：显示当前作答进度。
- 题目陈述文字。
- 李克特量表选项（由不同意到同意分级）。

**操作步骤：**
1. 阅读当前题目的陈述。
2. 点击最符合自身情况的选项。
3. 系统自动跳转至下一题。
4. 重复上述步骤，直至完成全部题目。

【配图：问卷作答界面（含进度条、题目、选项）】

### 4.3 提交并生成结果

完成全部题目后，系统自动计算五维人格得分并跳转至结果页。

【配图：作答完成/提交时的界面】

## 7. 单机对战（人机对战）

### 7.1 进入对战大厅并设置参数

从测评结果页点击「单机对战」，进入对战大厅。大厅用于设置本局参数。

**可设置项：**
- **AI 难度**：简单（凭直觉出牌）/ 中等（会记牌分析）/ 困难（推测你的心理）。
- **游戏轮数**：5 轮 / 10 轮 / 15 轮 / ∞（无限，直到有人胡）。
- **看牌难度**：明牌（全场人格公开，默认）/ 半公开（每回合查看 4 张并保留）/ 隐藏（每回合查看 2 张但不保留）。
- **对手档案**：展示三名 AI 对手（大雄 / 陈教授 / 老林）及其简介。

**操作步骤：**
1. 选择 AI 难度。
2. 选择游戏轮数。
3. 选择看牌难度。
4. 查看对手档案。
5. 点击「开始对战」按钮，进入游戏界面。

【配图：对战大厅界面（AI 难度 / 游戏轮数 / 看牌难度 / 对手档案）】

### 7.2 游戏主界面说明

进入对战后显示游戏主界面。

**界面区域说明：**
- **顶部对手区**：显示各 AI 对手的头像、手牌数量、已公开归档，当前出牌者会有高亮提示。
- **中央牌桌区**：包含抽牌堆（显示剩余张数）、弃牌堆（显示已弃张数）、行动记录。
- **底部玩家区**：显示本人手牌、五维归档进度、操作按钮（食胡、自摸碰等）、当前回合信息与记录入口。

【配图：单机对战主界面整体（标注对手区 / 牌桌区 / 玩家区）】

### 7.3 游戏操作流程

轮到本人回合时，按以下步骤操作：

**步骤一：抽牌**
- 点击中央「抽牌」牌堆，抽取一张牌。抽牌后进入出牌阶段。

【配图：点击抽牌堆抽牌的界面】

**步骤二：查看手牌（仅在半公开 / 隐藏难度下）**
- 在非明牌难度下，可点击「查看」按钮，翻看若干张手牌的人格维度。

【配图：查看手牌界面（半公开 / 隐藏难度下）】

**步骤三：归档（碰牌）**
- 当手中集齐同一人格维度、且数量达到该维度目标张数时，可进行归档。
- 点击「自摸碰」，选择要归档的维度，再选中对应的手牌，点击「归档判定」提交。
- 若归档正确，则完成该维度归档；若混入其他维度或重复归档，将被判失败并受罚（下一回合罚停）。

【配图：归档（碰牌）选牌与判定界面】

**步骤四：弃牌**
- 若本回合不归档，则需弃出一张牌。先圈定一张要舍弃的线索牌，再点击「提交弃牌」确认。

【配图：选牌弃牌与「提交弃牌」确认界面】

**步骤五：响应对手弃牌（碰 / 胡）**
- 当对手弃牌时，若可据此完成归档，界面会弹出判定窗口；可选择「碰」（选牌归档）、「食胡」或「过」。

【配图：响应对手弃牌的判定窗口】

**步骤六：食胡（获胜）**
- 当五个人格维度全部完成归档后，点击「食胡」宣告获胜。

【配图：食胡按钮 / 宣告获胜界面】

### 7.4 对局结算

一局结束后显示结算界面，展示各玩家名次、完成归档数与剩余手牌。

**操作步骤：**
1. 查看本局结算结果与名次。
2. 点击「再来一局」重新开始，或「返回主页」退出。

【配图：单机对战结算界面】

## 8. 联机对战（多人在线对战）

### 8.1 进入联机入口

从测评结果页点击「联机对战」，进入联机入口页。

**操作步骤：**
1. 学号已在测评时输入并保存在浏览器中（cookie），此处自动带入、无需再次输入。
2. 选择头像。
3. 选择人格牌堆（如 Big Five 五因素人格）。
4. 选择「创建房间」或「加入房间」。

【配图：联机入口页（学号已自动带入 / 头像 / 牌堆 / 创建或加入）】

### 8.2 创建房间（房主）

选择「创建房间」后设置房间参数并创建。

**可设置项：**
- 最多玩家数：2 / 3 / 4 人。
- 游戏轮数：∞ / 3 / 5 / 10。
- 看牌难度：明牌（全场人格公开，默认） / 半公开 / 隐藏。

**操作步骤：**
1. 设置最多玩家数、游戏轮数、看牌难度。
2. 点击「创建房间」。
3. 系统生成房间码，进入房间等待界面。

【配图：创建房间设置界面】

### 8.3 房间等待与邀请

房间创建后进入等待界面，显示房间码，供其他玩家加入。

**界面元素说明：**
- 房间码（可点击复制按钮复制）。
- 玩家列表（显示已加入玩家与空位）。
- 房间设置（只读展示：最多玩家数、游戏轮数）。
- 房主可见「开始对战」按钮（人数满足后可点击）；非房主显示「等待房主开始」。

**操作步骤：**
1. 将房间码分享给其他玩家（可点击复制按钮）。
2. 等待玩家加入，玩家列表实时更新。
3. 人数满足后，房主点击「开始对战」。

【配图：房间等待界面（房间码 / 玩家列表 / 房间设置 / 开始对战）】

### 8.4 加入房间（非房主）

其他玩家在联机入口页选择「加入房间」。

**操作步骤：**
1. 输入房主提供的房间码。
2. 点击加入，进入房间等待界面。
3. 等待房主开始对战。

【配图：加入房间（输入房间码）界面】

### 8.5 联机对战与结算

房主开始后，所有玩家进入联机对战界面，操作方式与单机对战一致（抽牌、查看、归档、弃牌、碰、食胡），游戏状态在各玩家间实时同步。对局结束后显示结算界面。
